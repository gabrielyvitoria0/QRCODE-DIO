QRCode Generator

Projeto desenvolvido durante meus estudos na DIO, focado em back-end e automações com Node.js.

✨ Sobre o projeto

Esse projeto gera QR Codes diretamente pelo terminal utilizando JavaScript e Node.js.

A ideia foi praticar:

manipulação de pacotes NPM
lógica back-end
automações simples
geração de QR Codes
estruturação de projetos
⚡ Funcionalidades

✔️ Gerador de QR Code
✔️ Execução via terminal
✔️ Conversão de links/textos
✔️ Projeto simples e prático para estudos

🚀 Executando o projeto

Clone o repositório:

git clone https://github.com/gabrielyvitoria0/QRCODE-DIO.git

Entre na pasta:

cd QRCODE-DIO

Instale as dependências:

npm install

Inicie o projeto:

node index.js
